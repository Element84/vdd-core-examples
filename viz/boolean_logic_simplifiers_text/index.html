About

This is for visualizing some code that simplifies a boolean condition tree. It currently only implements one simplification - the flattening of conditions from nested AND operations. This visualization is implemented using nested HTML Unordered Lists.

The code being visualized is located in vdd-core-examples/src/boolean_logic_simplifiers/. The visualization code is in vdd-core-examples/viz/boolean_logic_simplifiers_text/.
